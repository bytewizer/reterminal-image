# Yet Another Linux Distro - Build Repository

## Build

```
$ git clone https://github.com/meisteg/yald-build.git
$ cd yald-build
$ ./repo-mgr.sh install
$ ./repo-mgr.sh init
$ ./run-build.sh
```

### Supported machines

- raspberrypi3-64
